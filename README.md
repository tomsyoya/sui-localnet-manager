# SUI Localnet Manager

SUI Localnet Managerは、SUI testnet環境をGUIで管理するmacOSデスクトップアプリケーションです。開発者がコマンドライン操作なしでSUIテストネットワークの起動、停止、監視、設定を行えます。

## 概要

このアプリケーションは、SUIブロックチェーンの開発者向けに設計されたローカルネットワーク管理ツールです。直感的なGUIを通じて、SUIローカルネットワークの完全な制御と監視機能を提供します。

## 主な機能

### ネットワーク管理
- **ワンクリック起動・停止**: SUIローカルネットワークの簡単な起動と停止
- **リアルタイム監視**: ネットワーク状態、ノード数、ブロック高、トランザクション数の監視
- **プロセス管理**: 既存のSUIプロセスの自動検出と管理

### 設定管理
- **プロファイル管理**: 複数のネットワーク設定プロファイルの作成と管理
- **カスタム設定**: ポート番号、ノード数、SUIバイナリパスの設定
- **設定保存**: JSON形式での設定の永続化

### ログ・監視システム
- **リアルタイムログ**: SUIプロセスからのリアルタイムログストリーミング
- **ログフィルタリング**: レベル別（info、warn、error、debug）でのログフィルタリング
- **ログ検索**: キーワードによるログ検索機能
- **ログエクスポート**: ログのファイルエクスポート機能

### ユーザーインターフェース
- **Material-UI**: モダンで直感的なユーザーインターフェース
- **ダークテーマ対応**: システム設定に応じたテーマ切り替え
- **レスポンシブデザイン**: 様々な画面サイズに対応

## 技術仕様

### アーキテクチャ
- **フレームワーク**: Electron + TypeScript
- **フロントエンド**: React + Material-UI (MUI)
- **バックエンド**: Node.js (Electronメインプロセス)
- **IPC通信**: Electron IPCによるフロントエンド・バックエンド間通信

### 対応環境
- **OS**: macOS (主要対象)
- **SUIバージョン**: 最新の安定版SUI CLI
- **Node.js**: v16以上

### システム要件
- macOS 10.15以上
- 4GB以上のRAM
- SUI CLIがインストール済み、または手動でパス指定

## インストール

### 前提条件
SUI CLIをインストールしてください：
```bash
# Homebrewを使用
brew install sui

# または、Cargoを使用
cargo install --locked --git https://github.com/MystenLabs/sui.git --branch devnet sui
```

### アプリケーションのビルド
```bash
# リポジトリのクローン
git clone <repository-url>
cd sui-localnet-manager

# 依存関係のインストール
npm install

# アプリケーションのビルド
npm run build

# 開発モードでの実行
npm run dev

# 本番用パッケージの作成
npm run package
```

## 使用方法

### 初回起動
1. アプリケーションを起動
2. 設定画面でSUIバイナリのパスを確認・設定
3. プロファイルを作成（任意）
4. ダッシュボードからネットワークを起動

### 基本操作

#### ネットワークの起動
1. ダッシュボード画面の「ネットワーク開始」ボタンをクリック
2. Genesis作成とネットワーク起動の進行状況をログで確認
3. ネットワーク状態が「実行中」になることを確認

#### ネットワークの停止
1. 「ネットワーク停止」ボタンをクリック
2. プロセスの安全な終了を待機

#### プロファイル管理
1. プロファイル画面でカスタム設定を作成
2. ポート番号、ノード数などを指定
3. 設定を保存してダッシュボードで使用

#### ログ監視
1. ダッシュボード下部のリアルタイムログビューアで監視
2. ログ画面でより詳細な検索・フィルタリング
3. 必要に応じてログをエクスポート

## 設定ファイル

### アプリケーション設定
- **場所**: `~/Library/Application Support/SuiLocalnetManager/`
- **形式**: JSON
- **内容**: SUIパス、プロファイル、UI設定

### プロファイル設定
各プロファイルには以下の設定が含まれます：
- ネットワーク名
- ポート番号（デフォルト: 9000）
- ノード数（デフォルト: 4）
- Epoch時間
- その他のSUI起動オプション

## トラブルシューティング

### よくある問題

#### ネットワーク起動の失敗
- SUI CLIが正しくインストールされているか確認
- ポート競合がないか確認（他のSUIプロセスを停止）
- 設定画面でSUIバイナリパスが正しいか確認

#### ログが表示されない
- ネットワークが正常に起動しているか確認
- アプリケーションを再起動

#### パフォーマンスの問題
- 不要なログレベルをフィルタリング
- 長時間実行後はログをクリア

### ログファイル
- アプリケーションログ: `~/Library/Logs/SuiLocalnetManager/`
- SUI出力ログ: 作業ディレクトリの `sui_output.log`
